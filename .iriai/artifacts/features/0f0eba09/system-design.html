Overview

A single self-contained static HTML file (charmander.html) added to the project root alongside the existing index.html. No services, no databases, no APIs — purely a static file served by any web server or opened via file:// protocol.

Architecture Decisions

  1. Single flat color (#ff6600 orange on #1a1a1a dark) for all ASCII art text — matches the Pikachu page's single-color approach
  2. Mirror exact CSS structure from index.html (flexbox centering, Courier New monospace, 14px font, 1.2 line-height) — only background-color and color differ
  3. No navigation links between pages — pages are fully independent per REQ-9
  4. No JavaScript, no external dependencies — page is entirely self-contained per REQ-6
  5. Standard ASCII characters only — no Unicode art characters
