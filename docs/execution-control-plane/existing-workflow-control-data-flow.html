Durable Data And Side Effects

Postgres Artifacts

dag, dag-task:*, dag-verify:*, dag-commit-failure:*, dag-group:*, dag-gate:*, regroup markers.

Postgres Events

Phase events, agent starts/dones, verify failures, RCA/repair starts, commit failures, quiesce markers, Slack bridge status.

Feature Repositories

Canonical repos under the feature root are mutated directly by implementers, repair agents, commit helpers, and post-DAG gates.

Runtime Context

Model prompts draw from task specs, latest artifacts, selected file contents, verifier concerns, and generated context packages.

Authority Ambiguity

Task evidence can claim files that are stale, aliased, unmerged, uncommitted, or unverifiable in the canonical repo.
